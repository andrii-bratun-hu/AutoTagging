# Autotagging service

---
**Requirements**: 
- Python >= 3.6, 
- [other requirements](./requirements.txt)
- pip install -r requirements
---
**Run commands**: 
- Run locally:
  - python app.py
- Docker
  - docker build -t autotagging .  
  - docker run -p 8000:8000 autotagging
---
**Open dev tool**: 
- http://0.0.0.0:8000/docs
- http://0.0.0.0:8000/redoc
